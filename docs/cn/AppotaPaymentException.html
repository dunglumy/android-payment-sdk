AppotaPaymentException

package com.appota.core

public class AppotaPaymentException
extends Exception

概况

AppotaPaymentException是在实施支付交易过程中出现的exception 定义的class

方式

constructor AppotaPaymentException(String exceptionMessage)
用message 起造exception的函数
constructor AppotaPaymentException(String exceptionMessage, Throwable reason)
用massage 和reason 起造exception 的函数
constructor AppotaPaymentException(Throwable cause)
用cause 起造exception 的函数
