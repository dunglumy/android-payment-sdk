AppotaPaymentResult

package com.appota.payment.model

public class AppotaPaymentResult
extends Object
implements Serialize

概况

AppotaPaymentResult 是Appota Payment起造交易成功后的class callback

方式

String getTarget()
拿取返回的target
void setTarget(String target)
设定target
String getState()
拿取返回的state
void setState(String state)
设定state
String getTransactionId()
拿取返回的transactionId
void setTransactionId(transactionId)
设定transactionId
int getErrorCode()
拿取错误代码
void setErrorCode(int errorCode)
设定错误代码
String getErrorMessage()
拿取错误代码的公告
void setErrorMessage(String errorMessage)
设定错误代码的公告
