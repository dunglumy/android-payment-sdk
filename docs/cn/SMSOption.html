SMSOption

package com.appota.payment.model

public static class SMSOption
extends Object
implements Serialize

概况

SMSOption是短信支付方式中可以选择短信网关的class, 比如说：8488, 8588, 8688...

具体方式

String getSyntax()
拿取返回SMS的句法
void setSyntax(String syntax)
设定syntax
String getSendNumber()
拿取返回的短信网关
void setSendNumber()
设定sendNumber
double getAmount()
拿去返回的amount
void setAmount(double amount)
设定amount
