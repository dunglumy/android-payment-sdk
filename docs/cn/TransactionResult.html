TransactionResult

package com.appota.payment.model

public class TransactionResult
extends AppotaPaymentResult

概况

TransactionResult是交易成功后的class callback

具体方式

boolean isStatus()
返回的status (true/false)
void setStatus(boolean status)
设定status
String getAmount()
拿取返回的amount
void setAmount(String amount)
设定amount
String getCurrency()
拿取返回的currency
void setCurrency(String currency)
设定currency
String getType()
拿取返回的type
void setType(String type)
设定type
String getTime()
拿取返回的time
void setTime(String time)
设定time
