General

AppotaPayment is a class which executes all main features of Payment SDK. This class provides functions that call to payment interfaces for each various methods.

public void makeCardPayment(String cardSerial, String cardCode, String vendor, String noticeUrl, String state, String target, CardPaymentCallback callback) throws AppotaPaymentException

Payment by direct cards, do not use displayed interface of Appota SDK. Users are linked directly to the payment gateway's website via browser

Parameters:

- cardSerial: Serial Number
- cardCode: Card Code Number
- vendor: Suppliers. Now supporting:
        vinaphone: Vinaphone Card
        mobifone:  Mobifone Card
        viettel:       Viettel Card
        fpt:           FPT Gate Card
        mega:       MegaCard       
- noticeUrl: URL called when transaction ends
- state: A seri of customized states for dev to send to
- target: A seri of customized targets for dev to send to
- callback: Callback function called when transaction ends
