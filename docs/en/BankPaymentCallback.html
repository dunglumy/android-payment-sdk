BankPaymentCallback

package com.appota.payment.callback

public interface BankPaymentCallback

General

BankPaymentCallback is an interface defines callback functions after process of constructing Internet Banking transaction with Appota Payment System.

Methods

void onBankPaymentSuccess(BankPaymentResult result)
Callback function called when constructing a successful transaction.
void onBankPaymentError(String message)
Callback function calles when constructing a fail transaction
