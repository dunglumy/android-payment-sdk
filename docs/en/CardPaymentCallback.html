CardPaymentCallback

package com.appota.payment.callback

public interface CardPaymentCallback

General

CardPaymentCallback is an interface that defines callback functions after process of constructing Card transaction with Appota Payment System.

Methods

void onCardPaymentSuccess(CardPaymentResult result)
Callback functuon called when constructing a successful transaction.
void onCardPaymentError(String message)
Callback functuon called when constructing a fail transaction.
