SMSPaymentCallback

package com.appota.payment.callback

public interface SMSPaymentCallback

General

CardPaymentCallback is an interface that defines callback functions after process of constructing SMS transaction with Appota Payment system.

Methods

void onSMSPaymentSuccess(SMSPaymentResult result)
Callback function called when constructing a successful transaction.
void onSMSPaymentError(String message)
Callback function called when constructing a fail transaction.
