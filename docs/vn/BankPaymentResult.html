BankPaymentResult

package com.appota.payment.model

public class BankPaymentResult
extends AppotaPaymentResult

Tổng quan

BankPaymentResult là class định nghĩa kết quả trả về cho callback sau khi hệ thống Appota Payment khởi tạo thanh toán internet banking thành công.

Phương thức

List<BankOption> getBankOptions()
Lấy danh sách các tùy chọn cổng thanh toán được trả về. Với mỗi tùy chọn ngân hàng thì có thể có một hoặc nhiều cổng thanh toán phù hợp.
void setBankOptions(List<BankOption> bankOptions)
Đặt giá trị bankOptions
