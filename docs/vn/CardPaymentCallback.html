CardPaymentCallback

package com.appota.payment.callback

public interface CardPaymentCallback

Tổng quan

CardPaymentCallback là interface định nghĩa các hàm callback sau quá trình thực hiện khởi tạo giao dịch thẻ cào với hệ thống Appota Payment.

Phương thức

void onCardPaymentSuccess(CardPaymentResult result)
Hàm callback được gọi khi khởi tạo giao dịch thành công.
void onCardPaymentError(String message)
Hàm callback được gọi khi khởi tạo giao dịch thất bại.
