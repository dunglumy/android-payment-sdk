PaypalPaymentCallback

package com.appota.payment.callback

public interface PaypalPaymentCallback

Tổng quan

PaypalPaymentCallback là interface định nghĩa các hàm callback sau quá trình thực hiện khởi tạo giao dịch Paypal với hệ thống Appota Payment.

Phương thức

void onPaypalPaymentSuccess(PaypalPaymentResult result)
Hàm callback được gọi khi khởi tạo giao dịch thành công.
void onPaypalPaymentError(String message)
Hàm callback được gọi khi khởi tạo giao dịch thất bại.
