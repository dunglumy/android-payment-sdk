SMSOption

package com.appota.payment.model

public static class SMSOption
extends Object
implements Serialize

Tổng quan

SMSOption là class định nghĩa tùy chọn sử dụng đầu số SMS trong thanh toán bằng hình thức SMS. VD: 8488, 8588, 8688...

Phương thức

String getSyntax()
Lấy cú pháp SMS trả về
void setSyntax(String syntax)
Đặt giá trị trường syntax
String getSendNumber()
Lấy giá trị đầu số trả về
void setSendNumber()
Đặt giá trị trường sendNumber
double getAmount()
Lấy giá trị trường amount trả về
void setAmount(double amount)
Đặt giá trị trường amount
