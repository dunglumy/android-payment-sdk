TransactionStatusCallback

package com.appota.payment.callback

public interface TransactionStatusCallback

Tổng quan

TransactionStatusCallback là interface định nghĩa các hàm callback sau quá trình thực hiện giao dịch thanh toán.

Phương thức

void onTransactionSuccess(TransactionResult result)
Hàm callback được gọi khi giao dịch thành công.
void onTransactionError(String message)
Hàm callback được gọi khi giao dịch thất bại.
